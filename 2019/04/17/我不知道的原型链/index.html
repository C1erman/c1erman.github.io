首先，请再次明确这个基本概念：函数也是对象。

对象

概括地说，JavaScript中的对象是从字符串到属性的映射。这也难怪不少人将对象用于Map结构的简单实现。

对象的属性类型

一个对象可以拥有许多属性：自有属性、继承属性、内置属性等等。而他们可以被分为三类：

  • （一般）属性：我们通常所做的、信手为对象所添加的属性，属性的值可以是任意JavaScript值
  • 访问器：访问器是一个方法，它提供了在用户读取、写入一个对象的属性时可以计算属性的值的权力
  • 内置属性：只存在于ECMAScript语言规范中，规范将内置属性的键置于方括号中，比如Object[[Prototype]]

属性特性与属性描述符

所有的属性状态，包括它的数据，都存储在特性中。它们是属性拥有的字段，就像对象拥有的属性。特性的键通常写在双方括号中，特性与普通属性和访问器getter、setter有关。

普通属性拥有以下特性：

  • [[Value]]持有属性的值，即它的数据
  • [[Writable]]持有布尔类型的值，表示属性值是否可以改变

访问器具有以下特性：

  • [[Get]]持有getter，读取属性时调用的函数，该函数计算读取的结果
  • [[Set]]持有setter，为属性设置时调用的函数，该函数接收设置的值作为参数

所有的属性都有以下特性：

  • [[Enumerable]]持有一个布尔值，设置一个属性是否可以枚举，如果不可枚举，则会在一些操作中忽略这个属性
  • [[Configurable]]持有一个布尔值，如果它是false，那么你不能删除、改变属性的任何特性，除了[[Value]]

如果没有指定特性，属性的特性的默认值如下：

属性键 默认值
[[Value]] undefined
[[Get]] undefined
[[Set]] undefined
[[Writable]] false
[[Enumerable]] false
[[Configurable]] false
