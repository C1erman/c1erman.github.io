Python备考知识点

发表于 2019-05-30 | 分类于 -备考

嗯，选了这门课就要好好准备考试啊。

数据类型

基本类型

  1. Number：数字类型

Python 2中没有bool类型，而Python 3中添加了这一类型，但仍用数字表示，True为1，False为0，因此bool类型能够与数字做运算。

  1. String：字符串类型

使用格式：string[start : end]来左闭右开地截取字符串。

索引值以0起始，-1为字符串末尾。

+运算连接字符串，*运算重复字符串。

  1. List：列表

使用格式：list[start : end : step]来左闭右开地截取列表，如果步长为负数，则代表逆向读取。

+运算符连接列表，*运算符重复列表。

  1. Tuple：元组

使用格式：tuple[start : end]来左闭右开地截取元组。

+运算符连接元组，*运算符重复元组。

  1. Set：集合

集合内不含重复的元素。

可以利用-、|、&、^来实现集合之间的差集、并集、交集和两集合中不同时存在的元素集。

  1. Dictionary：字典

使用dict()构造函数，从键值对序列中构造出字典。

当键名重复时，取最后一次的值。

注意点

不可变数据：数字、字符串、元组

可变数据：列表、集合、字典

不可变数据：

无法通过下标来对值进行改变。

赋予另一个新值时，会创建一个新值，并将变量名指向该值的物理地址。

可变数据：

可以通过下标对值改变。

对值改变时，变量名指向的物理地址不变。

字符串、列表、元组都属于序列。

类型转换

  • int(x[, base])

将x转换为一个整数。

  • float(x)

将x转换为一个浮点数。

  • eval(string)

计算字符串中的有效Python表达式，并返回一个对象。

  • tuple(s)

将序列s转为一个元组。

  • list(s)

将序列s转为一个列表。

  • set(s)

将序列s转为一个集合。

  • dict(d)

创建一个字典，d必须是一个key, value格式的序列。

流程控制

if-elif-else

Python使用elif代替了else-if。

while

for…in

Python中的for循环可以遍历任何序列的项目。

搭配range(start, end, step)：

循环时尝试用range函数，左闭右开地返回一个步长为step的数字列表。

函数

函数定义

使用def关键字来定义一个函数。

若是没有return语句，则被视为返回None。

函数传参

可以给函数的参数以默认值。

可以使用一个*号标识一个可变长度、元组类型的参数；两个*号标识一个长度可变、字典类型的参数。

单独出现的*号之后的参数必须明确指定。

匿名函数

使用lambda关键字实现一个匿名函数。我还是更喜欢箭头函数

该函数只包含一个语句！

格式为：lambda [arg1[, arg2]] : statement。

变量作用域

貌似？Python的变量作用域同JavaScript一样，是函数级作用域，但不同的是，Python中的变量必须先声明后使用。

当内部作用域想修改外部作用域的变量时，使用global关键字。
