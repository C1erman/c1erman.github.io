Python简易知识点

发表于 2019-05-30 | 更新于 2019-08-05 | 分类于 -烂笔头

对Python的简单总结。

数据类型

基本类型

  1. Number：数字类型

Python 2中没有bool类型，而Python 3中添加了这一类型，但仍用数字表示，True为1，False为0，因此bool类型能够与数字做运算。

复数的实部和虚部均为浮点数类型。

大小在-10308到10308之间，精度数量级为10-16。

此外，存在着四种进制的整数表示。

  1. String：字符串类型

三引号字符串与三引号注释：

使用#来单行注释，使用三个单、双引号来多行注释，但本质上，多行注释仍然是字符串。

使用格式：string[start : end]来左闭右开地截取字符串。

索引值以0起始，-1为字符串末尾。

+运算连接字符串，*运算重复字符串。

Unicode：

Python 3中使用Unicode码来表示字符串，'A'的Unicode码为65，'a'的为97，0的为48。

字符串以'\0'结尾，对应的Unicode码为0。

  1. List：列表

使用格式：list[start : end : step]来左闭右开地截取列表，如果步长为负数，则代表逆向读取。

+运算符连接列表，*运算符重复列表。

  1. Tuple：元组

使用格式：tuple[start : end]来左闭右开地截取元组。

+运算符连接元组，*运算符重复元组。

  1. Set：集合

集合内不含重复的元素，并且集合内只能存在不可变数据。

可以利用-、|、&、^来实现集合之间的差集、并集、交集和两集合中不同时存在的元素集。

  1. Dictionary：字典

使用dict()构造函数，从键值对序列中构造出字典。

当键名重复时，取最后一次的值。

注意点

不可变数据：数字、字符串、元组

可变数据：列表、集合、字典

不可变数据：

无法通过下标来对值进行改变。

赋予另一个新值时，会创建一个新值，并将变量名指向该值的物理地址。

可变数据：

可以通过下标对值改变。

对值改变时，变量名指向的物理地址不变。

字符串、列表、元组都属于序列。

类型转换

  • int(x[, base])

将x转换为一个整数。

  • float(x)

将x转换为一个浮点数。

  • eval(string)

计算字符串中的有效Python 3表达式，并返回一个对象。

  • tuple(s)

将序列s转为一个元组。

  • list(s)

将序列s转为一个列表。

  • set(s)

将序列s转为一个集合。

  • dict(d)

创建一个字典，d可以是一个元组或列表，d的内容为key, value元组或列表对。

操作符

Python中提供了一系列的操作符用于简化运算。

del

del操作符删除不再使用的可变数据或整个变量。

is

判断两个值是否相等。

对于可变数据类型：按引用比较。

对于不可变数据类型：按值比较。

原来你是JavaScript。

in

判断一个值是否存在于序列、集合、字典中。

not

非。

也可搭配in使用，将值取反。

| or

或。

也可用于集合的并集。

& and

与。

也可用于集合的交集。

流程控制

if-elif-else

Python使用elif代替了else-if。

while

for…in

Python中的for循环可以遍历任何序列的项目。

搭配range(start, end, step)：

循环时尝试用range函数，左闭右开地返回一个步长为step的数字列表。

函数

函数定义

使用def关键字来定义一个函数。

若是没有return语句，则被视为返回None。

函数传参

可以给函数的参数以默认值。

可以使用一个星号*标识一个可变长度、元组类型的参数；两个星号**标识一个长度可变、字典类型的参数。

可变长度参数不能位于函数参数中的第一位。

单独出现的*号之后的参数必须明确指定。

匿名函数

使用lambda关键字实现一个匿名函数。我还是更喜欢箭头函数

该函数只包含一个语句！

格式为：lambda [arg1[, arg2]] : statement。

变量作用域

貌似？Python的变量作用域同JavaScript一样，是函数级作用域，但不同的是，Python中的变量必须先声明后使用。

当内部作用域想修改外部作用域的变量、或想临时定义全局变量时，使用global关键字。

异常捕获

Python 3使用关键字try、except、else、finally进行异常捕获与处理。

输入与输出

输入

使用函数input(string)，该函数返回用户输入的字符串。

输出

使用print(something, end)函数输出内容。

end参数表明该输出结束之后以什么字符结尾，默认值为换行符'\n'，也常使用回车符'\r'实现输出覆盖。

通常该函数与string.format(number | string)搭配使用，该函数会填充字符串中的每一个插槽{}，插槽下标从0开始。

插槽内部的格式为：{<参数序号>:<格式化控制标记>}，其中，引号:为引导符号。

格式化控制标记的其作用如下表：

控制标记 说明 取值 示例
<填充> 用于填充的单个字符 默认值为' ' '{0:x^10}'.format(1) //'xxxx1xxxxx'
<对齐> 控制填充时字符对齐方式 <左对齐、>右对齐、^居中对齐 '{0:^10}'.format(1) //' 1 '
<宽度> 设置插槽的输出宽度 整数 '{:^10}'.format(1) //' 1 '
<,> 是否显示数字的千分位分隔符 默认不显示 '{:,}'.format(1000) //'1,000'
<.数字精度> 浮点数的小数精度 整数 '{:.5f}'.format(1) //'1.00000'
<类型> 数字类型转换 b二进制、f浮点数、e科学计数法 '{:e}'.format(3.1) //'3.100000e+00'

再次注意，format()函数既可以格式化字符串，又可以用来格式化数字。

文件读写

文件类型

Python 3中，一般将文件分为文本文件与二进制文件。

文本文件由单一的特定编码组成，常见后缀为.txt、.js、.py。

二进制文件由0、1组成，常见后缀为.png、.avi、.bmp。

打开文件

使用函数open(filePath, mode)以模式为mode的方法打开一个文件，并创建一个file对象。

filePath为文件的绝对或相对路径。

注意：

与其它语言的转换方式一样，绝对路径中的单反斜杠\要转为双斜杠//。

常见的mode如下表所示：

mode 説明
t 文本模式，为默认值
b 以二进制模式打开文件
x 以只写方式创建文件，如果文件已经存在则会报错。
w 以只写方式打开文件，如果文件已存在则打开文件，并从头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。
a 打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。
w+ 打开一个文件用于读写。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。
a+ 打开一个文件用于读写。如果该文件已存在，文件指针将会放在文件的结尾。文件打开时会是追加模式。如果该文件不存在，创建新文件用于读写。

文件指针位置

  • file.seek(offset[, whence])

将文件读取指针移动到指定的位置。

offset为偏移量，将要移动的字节数。

whence为偏移量的参考点，0为默认值，代表从文件开头开始算起；1代表从当前位置开始算起；2代表从文件末尾开始算起。

  • file.tell()

返回文件当前指针位置。

读

常用的file读API如下：

  • file.read(size)

从文件读取指定的字节数，未给定或为负时读取全部。

  • file.readline(size)

从文件读取整行，包括'\n'字符，size参数起相同的作用。

  • file.readlines(lineSize)

从文件读取lineSize的行，以列表形式返回这些行。

写

常用的file写API如下：

  • file.write(string)

向文件写入指定字符串。

  • file.wirtelines(list)

向文件写入由迭代对象产生的序列字符串。

一般传入的参数为字符串列表。

关闭文件

使用函数file.close()关闭一个打开的文件。

如果不关闭，该文件将被占用而无法再次打开。
